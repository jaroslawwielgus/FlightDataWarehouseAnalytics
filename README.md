# Data Warehouse and Analytics about flights

## What was done
- engineered a data warehouse by extracting and processing aviation .csv datasets from Kaggle
- created a UML conceptual model using Visual Paradigm and developed an OLAP cube using SSAS in Visual Studio
- designed and orchestrated end-to-end ETL pipelines (Control and Data Flows) using SSIS in Visual Studio
- conducted a comprehensive data analysis using Tableau and Excel, building charts to visualize key business metrics, including travel seasonality, pricing trends and passenger volume distribution.
